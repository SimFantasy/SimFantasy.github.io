Styled-components中使用媒体查询

Simz
2022-10-29
react styled-components

styled-components和emotion中作媒体查询很麻烦，不如scss中的mixin来的简单。从emotion上找到一个包facepaint，能在一定程度上优化代码，减少代码量，但也不能像正常写css那样去写样式了，只能用驼峰的方式写成对象形式的样式。我们来分别看下这三种方式的媒体查询：

‍

Scss

 1// scss/_breakproint.scss
 2
 3@mixin mobile {
 4  @media only screen and (max-width: 575.98px) {
 5    @content;
 6  }
 7}
 8
 9@mixin tablet {
10  @media only screen and (max-width: 991.98px) {
11    @content;
12  }
13}
14
15// app.scss
16@use "./scss/" as *;
17
18body {
19  @include tablet {
20    font-size: 80%;
21  }
22  @include mobile {
23    font-size: 60%;
24  }
25}

‍

Styled-components

 1// styles/variable.js
 2
 3const breakpoints = [576, 992]
 4const mq = breakpoints.map(bp => `@media (min-width: ${bp}px)`)
 5
 6export default { mq }
 7
 8
 9// app.js
10import styled from 'styled-components'
11import v from '@/styles/variable'
12
13const AppWrap = styled.div`
14  width: 100vw;
15  height: 100vh;
16  ${{
17    [v.mq[0]]: {
18      fontSize: '40px'
19    },
20    [v.mq[1]]: {
21      fontSize: '60px'
22    }
23  }}
24`

‍

Facepaint

首先需要安装下包

1pnpm add facepaint

使用

 1// src/styles/variable.js
 2
 3import facepaint from 'facepaint'
 4
 5const mediaQuery = facepaint([
 6  '@media(max-width: 991.98px)',
 7  '@media(max-width: 575.98px)'
 8])
 9
10// app.js
11import styled from 'styled-components'
12import v from '@/styles/variable'
13
14const AppWrap = styled.div`
15  width: 100vw;
16  height: 100vh;
17  ${v.mediaQuery({
18    backgroundColor: ['#fff', '#ccc', '#000']
19  })};
20`

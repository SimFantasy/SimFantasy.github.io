JavaScript字符串操作整理

Simz
2022-04-29
javascript

基础差就只能慢慢补，这两天把字符串操作的方法整理了下，平常用的也就只有几个，但是一整理就发现有一大堆。

常用indexOf()、split()、slice()、toLowerCase()、toUpperCase()、replace()、trim()、padStart()、parseInt()

获取字符串长度

获取字符串长度 length

1const str = 'Hello World'
2str.length // 11

获取字符串指定位置的值

获取指定位置的字符 charAt()

1const str = 'Hello World'
2str.charAt(1) // e

获取指定位置的Uniconde值 charCodeAt()

1const str = 'Hello World'
2str.charCodeAt(1) // 117

检索字符串是否包含特定序列

查找某个字符 IndexOf()

有则返回第一次匹配到的索引位置index，否则返回-1

1const str = 'Hello World'
2str.indexOf('l') // 2
3str.indexOf('x') // -1

查找某个字符 lastIndexOf()

有则返回最后匹配到的索引位置index ，否则返回-1 。只是查找的顺序不一样，indexOf()是正序查找，lastIndexOf()是逆序查找。

1const str = 'Hello World'
2str.lastIndexOf('l') // 8
3str.lastIndexOf('y') // -1

判断字符串是否包含指定子字符串 includes()

如找到匹配字符串返回true否则返回false 。

参数：

searchvalue，必须，要查找的字符串。

start，开始查找位置，默认为0。

1const str = 'Hello World'
2str.includes('o') // true
3str.includes('x') // false
4str.includes('e', 2) // false

检测字符串是否以指定的子字符串开始 startsWith()

如以指定子字符串开始返回true，否则返回false 。

1const str = 'Hello World'
2str.startsWith('He') // true
3str.startsWith('he') // false
4str.startsWith('Wo', 6) // true

检测字符串是否以指定的字符串结尾 endsWith()

如以指定字符串结尾返回true，否则返回false。

参数：

searchvalue，必须，要查询的字符串。

length，查询字符串的长度范围，默认为整个字符串长度范围内。

1const str = 'Hello World'
2str.endsWith('ld') // true
3str.endsWith('ed') // false
4str.endsWith('llo', 5) // true

连接多个字符串

连接两个或多个字符串 concat()

不会改变原有字符串，会返回连接后的新的字串

1const str = 'Hello World'
2str.concat(' Javascript') // Hello World Javascript
3str.concat(' Vue', ' React') // Hello World Vue React

字符串分割成数组

字符串分割成字符串数组 split()

不会改变原始字符串。

参数：

separator，必须，字符串或正则表达式，从该参数指定的地方分割字符串。

limit，可选，指定返回数组中字符串长度。未设置该参数则整个字符串都会被分割。

1const str = 'Hello World'
2str.split('') // //[ 'H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd' ]
3str.split('', 5) // [ 'H', 'e', 'l', 'l', 'o' ]
4str.split(' ') // ['Hello', 'World']

截取字符串

提取字符串某个部分 slice(）

以新的字符串返回被提取的部分。该方法返回的字符串：包括开始处的字符，不包括结束处的字符 。

参数：

start，必须，要截取片段开始下标，第一个字符位置为0，如为负数，则从未必开始截取。

end，可选，要截取片段结尾下标。若未指定该参数，则从start到原始字符串结尾的字符串。若参数为负数，则从字符串尾部开始算起。

1const str = 'Hello World'
2str.slice(1, 4) // ello
3str.slice(6) // World
4str.slice() // Hello World
5str.slice(-2) // ld
6str.slice(6, 1) //

提取两个指定下标之间的的字符 substring()

MDN建议使用substring()替代substr()。 返回一个新的字符串。

参数：

from，必须，非负整数，规定要提取子字符串的起始位置。

to，可选，非负整数，规定要提取子字符串的结束位置。如省略，则返回的字符串一直到字符串的结尾。

注意：

  1. 如from和to相等，那么返回一个空串。
  2. 如from比to大，则会交换这两个参数，该方法不接受负数作为参数。
  3. 该方法返回的字符串：包括开始处的字符，不包括结束处的字符 。
1const str = 'Hello World'
2str.substring(0, 5) // Hello
3str.substring(6) // World
4str.substring() // Hello World
5str.substring(6, 0) // Hello
6str.substring(-1) // Hello World

字符串大小写转换

把字符串转换为小写 toLowerCase()

1const str = 'Hello World'
2str.toLowerCase() // hello world
3// 首字母变成小写
4console.log(str[0].toLowerCase() + str.substring(1)) // hello World

把字符串转换为大写 toUpperCase()

1const str = 'Hello World'
2str.toUpperCase() // HELLO WORLD

字符串匹配模式

替换字符 replace()

参数：

searchvalue，必须，规定子字符串或要替换的模式的RegExp对象。

newvalue，必须，字符串，规定了替换文本或生成文本的函数。

1const str = 'Hello World'
2str.replace('World', 'Javascript') // Hello Javascript
3str.replace(/world/gi, 'Vue') // Hello Vue

匹配字符串 match()

匹配字符串，查找到返回一个数组对象，数组对象里有 匹配的值 查找字符串的下标 和 原字符串内容，找不到返回null。

该方法参数regexp是必须的，规定要匹配的模式的RegExp对象，如果不是RegExp对象，则需要先传递给RegExp构造函数，将其转换为RegExp对象。

该方法返回存放匹配结果的数组。

1const str = 'Hello World'
2str.match('o') // [ 'o', index: 4, input: 'Hello World', groups: undefined ]
3str.match('f') // null

检索字符串 search()

跟indexOf一样，查找字符串首次出现的位置，如果找到返回该字符串下标，下标从0开始，未找到返回-1

1const str = 'Hello World'
2str.search('o') // 4
3str.search('f') // -1

移除字符串中的空白字符

移除首尾空白字符 trim()

不改变原字符串，返回一个新字符串。不适用于null、nudefined、Number类型。

1const str = '   Hello World   '
2str.trim() // Hello World

移除开头的空白字符 trimStart()

1const str = ' Hello World '
2str.trimStart() // Hello World

移除结尾的空白字符 trimEnd()

1const str = ' Hello World '
2str.trimEnd() // Hello World

获取字符串本身

返回字符串的原始值 valueOf()

1const str = 'Hello World'
2str.valueOf() // Hello World

转换成字符串类型 tostring()

一般用于将数值、布尔值类型转换成字符串类型。

1const str = 123
2str.toString() // "123"

重复字符串

重复字符串 repeat()

返回一个新字符串，重复n次。

  1. 如参数为0、负小数、NaN、字符串，输出为空
  2. 如为负数、infinity 会报错
  3. 如为小数，会向下取整
  4. 如果字符串会先转为数字
1const str = 'Hello World'
2str.repeat(2) // Hello WorldHello World
3str.repeat(0) //
4str.repeat(2.8) // Hello WorldHello World
5str.repeat(-1) // Error

补齐字符串长度

补齐头部字符串 padStart()

参数：

参数1，必须，数字，字符串补齐后的长度。如原字符串长度等于或大于指定长度，则返回原字符串

参数2，可选，字符，用来补全的字符串。如为空则用空格补全长度

1const str = 'Hello World'
2str.padStart(15, 'a') // aaaaHello World
3str.padStart(8, 'a') // Hello World
4str.padStart(15)  //    Hello World

补齐尾部字符串 padEnd()

参数同padStart。

1const str = 'Hello World'
2str.padEnd(15, 'a') // Hello Worldaaaa
3str.padEnd(3, 'a')
4str.padEnd(15)

字符串转为数字

字符串转为整数 parseInt()

参数：

string，必须，要被转换的字符串。

radix，可选，要解析的数字技术，介于2 ~ 36之间。

  1. 当参数 radix 的值为 0，或没有设置该参数时，parseInt() 会根据 string 来判断数字的基数。
  2. 当参数 radix 的值以 “0x” 或 “0X” 开头，将以 16 为基数
  3. 如果该参数小于 2 或者大于 36，则 parseInt() 将返回 NaN
  4. 只有字符串中的第一个数字会被返回，当遇到第一个不是数字的字符为止
  5. 如果字符串的第一个字符不能被转换为数字，就会返回 NaN
  6. 字符串开头和结尾的空格是允许的
1const numStr = "123"
2parseInt(numStr) // 123

字符串转为浮点数 parseFloat()

该方法指定字符串中的首个字符是否是数字。如果是，则对字符串进行解析，直到到达数字的末端为止，然后以数字返回该数字，而不是作为字符串。

1const numStr = "123"
2parseFloat(numStr)

ChakraUI 暗黑模式切换

Simz
2022-12-03
react chakraui

使用ChakraUI可以快速切换暗黑模式。并且可以用来做主题切换，提供了一系列的钩子和函数使用起来非常方便。

‍

安装

按官方给的安装，在vite中

1pnpm add @chakra-ui/react @emotion/react@^11 @emotion/styled@^11 framer-motion@^6

‍

配置

安装好了后，需要在App文件中配置下。

 1import React from 'react'
 2import { ChakraProvider, CSSReset } from '@chakra-ui/react'
 3import { BrowserRouter } from 'react-router-dom'
 4import RootRoutes from '@/router'
 5import Layout from '@/components/layout'
 6import RootStyle from '@/styles/base.style'
 7
 8const App = () => {
 9  return (
10    <ChakraProvider>
11      <CSSReset />
12      <RootStyle />
13      <BrowserRouter>
14        <Layout>
15          <RootRoutes />
16        </Layout>
17      </BrowserRouter>
18    </ChakraProvider>
19  )
20}
21
22export default App

‍

如何使用

配置完成后，我们就可以使用ChakraUI提供的钩子函数进行颜色主题的切换了

 1// src/pages/Home/index.jsx
 2
 3import React from 'react'
 4import {
 5  useColorMode,
 6  Button,
 7  Text,
 8  Box
 9} from '@chakra-ui/react'
10
11const Home = () => {
12  const { colorMode, toggleColorMode } = useColorMode()
13  return (
14    <div>
15      <Box w={400} h={200} bgColor={colorMode === 'light' ? 'blue.200' : 'red.100'}>
16        <Text>当前模式：{colorMode}</Text>
17      </Box>
18
19      <Button onClick={toggleColorMode}>切换模式</Button>
20    </div>
21  )
22}
23
24export default Home

这样就实现了最简单的一个明亮和暗黑主题切换的模式了。

但是这样有一个问题，如果每次都用三元表达式来写这种样式判断就太麻烦了，需要自己写个方法。ChakraUI当然也想到了这个问题，于是他们提供了一个useColorModeValue的钩子方法。

 1// src/pages/Home/index.jsx
 2
 3import React from 'react'
 4import {
 5  useColorMode,
 6  Button,
 7  Text,
 8  Box,
 9  useColorModeValue,
10  LightMode,
11  DarkMode
12} from '@chakra-ui/react'
13
14const Home = () => {
15  const { colorMode, toggleColorMode } = useColorMode()
16  const bgColor = useColorModeValue('blue.200', 'red.100')
17  return (
18    <div>
19      <Box w={400} h={200} bgColor={bgColor}>
20        <Text>当前模式：{colorMode}</Text>
21      </Box>
22
23      <Button onClick={toggleColorMode}>切换模式</Button>
24    </div>
25  )
26}
27
28export default Home

我们可以使用这个钩子方法来处理需要切换的颜色样式。

‍

强制主题模式样式

有的场景需要保持主题模式下处于明亮或暗黑样式不变化，我们就可以使用LightMode和DarkMode两个组件对于不需要变化的组件进行包裹。

 1// src/pages/Home/index.jsx
 2
 3import React from 'react'
 4import {
 5  useColorMode,
 6  Button,
 7  Text,
 8  Box,
 9  useColorModeValue,
10  LightMode,
11  DarkMode
12} from '@chakra-ui/react'
13
14const Home = () => {
15  const { colorMode, toggleColorMode } = useColorMode()
16  const bgColor = useColorModeValue('blue.200', 'red.100')
17  return (
18    <div>
19      <Box w={400} h={200} bgColor={bgColor}>
20        <Text>当前模式：{colorMode}</Text>
21        <Button>变色的按钮</Button>
22        <LightMode>
23          <Button colorScheme='purple'>不变色的按钮</Button>
24        </LightMode>
25        <DarkMode>
26          <Button colorScheme='orange'>不变色的按钮</Button>
27        </DarkMode>
28      </Box>
29
30      <Button onClick={toggleColorMode}>切换模式</Button>
31    </div>
32  )
33}
34
35export default Home

‍

设置默认主题模式

可以根据自定义主题的一些配置，来设置默认主题的模式。

通过extendTheme这个方法来配置继承的主题配置。

initialColorMode：设置默认主题模式，默认值是light。

useSystemColorMode：设置默认使用系统主题哦是，默认值是false。

 1// src/app.js
 2
 3import React from 'react'
 4import { ChakraProvider, CSSReset, extendTheme } from '@chakra-ui/react'
 5import { BrowserRouter } from 'react-router-dom'
 6import RootRoutes from '@/router'
 7import Layout from '@/components/layout'
 8import RootStyle from '@/styles/base.style'
 9
10const theme = extendTheme({
11  config: {
12    initialColorMode: 'dark'
13    // useSystemColorMode: true
14  }
15})
16
17const App = () => {
18  return (
19    <ChakraProvider theme={theme}>
20      <CSSReset />
21      <RootStyle />
22      <BrowserRouter>
23        <Layout>
24          <RootRoutes />
25        </Layout>
26      </BrowserRouter>
27    </ChakraProvider>
28  )
29}
30
31export default App

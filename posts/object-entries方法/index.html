Object.entries方法

Simz
2022-08-08
react javascript

最近在做Mac版的Netease Music的项目，在参考大佬的源码的时候看到了这个方法Object.entries，感觉甚妙，故记录之。

在项目中，有个歌单分类，返回的数据中，一组为一级分类的对象，一组为二级分类的数组。二级分类上有一级分类的值。

参考地址：歌单分类API

一级分类为对象，需要遍历出key和value，使用Object.entries就可以拿到key和value，并map出所有的一级分类，然后通过key，filter出二级分类中相应的子分类，并map出来。

代码如下：

 1const DEFAULT_CAT = '全部'
 2const [currentCat, setCurrentCat] = useState(selectedCat || DEFAULT_CAT)
 3const handleCatClick = (cat: string) => {
 4  setCurrentCat(cat)
 5  onCatSelect(cat)
 6}
 7
 8const renderCats = () => {
 9  return (
10  	<div className='popover'>
11    	<div className='all'>
12      	<span 
13          className={currentCat === DEFAULT_CAT ? 'active' : ''} 
14          onClick={() => handleCatClick(DEFAULT_CAT)}>
15          全部歌单
16        </span>
17      </div>
18      <div className="content">
19      	{Object.entries(cats.categories).map(([key, value]) => {
20          const subCats = cats.sub.filter(({category}) => category === Number(key))
21          return (
22            <div className='cat-block' key={key}>
23              <div className='label'>{value}</div>
24              <div className='content'>
25                {subCats.map(({name}) => (
26                  <div 
27                    key={name} 
28                    className={currentCat === name ? 'active': ''}
29                    onClick={() => handleCatClick(name)}
30                  >{name}</div>
31                ))}
32              </div>
33            </div>
34          )
35        }}
36      </div>
37    </div>
38  )
39}

顺便学习下Object.entries相关的方法Object.keys和Object.values

entries可以将key和value同时取出。

keys则只是取出key

values取出value

以后遇到对象转数组就不用怕了~

 1import React from 'react'
 2const categories = {
 3  0: '语种',
 4  1: '风格',
 5  2: '场景',
 6  3: '情感',
 7  4: '主题'
 8}
 9export default function ObjectFunc() {
10  const keys = Object.keys(categories)
11  const values = Object.values(categories)
12  const entriesMap = Object.entries(categories)
13  return (
14    <div>
15      ObjectFunc
16      <div>
17        <span>Keys: </span>
18        {keys.map((item, index) => (
19          <div key={index}>{item}</div>
20        ))}
21      </div>
22      <div>
23        <span>Values: </span>
24        {values.map((item, index) => (
25          <div key={index}>{item}</div>
26        ))}
27      </div>
28      <div>
29        <span>Entries: </span>
30        {entriesMap.map(([key, value]) => (
31          <div key={key}>{value}</div>
32        ))}
33      </div>
34    </div>
35  )
36}

相反，还有个数组转对象的方法Object.fromEntries，和Object.entries一起看下就明白了

1const object = { key1: 'value1', key2: 'value2' }
2
3const array = Object.entries(object)
4// [ ["key1", "value1"], ["key2", "value2"] ]
5
6Object.fromEntries(array)
7// { key1: 'value1', key2: 'value2' }

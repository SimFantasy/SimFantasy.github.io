React + Vite + ESLint + Prettier

Simz
2022-07-25
Vscode Vite Elint Prettier

Vite到3了，在React中比自带的create-react-app，快太多，用上就基本换不过去的感觉。记录下配置ESLint和Prettier，看了很多教程，配置大多都是和Typescript一起，本人还未到那个层级，只能上JS。废话少说开始干活~

创建项目

1yarn create vite

然后提示输入项目名称 my-app ，进入项目 cd my-app，安装依赖 yarn，然后启动项目yarn dev。运行后，vite项目的默认端口改成了5173。

ESLint

 1npm init @eslint/config
 2
 3// How would you like to use ESLint? Yes
 4// What type of modules does your project use? esm
 5// Which framework does you project use? react
 6// Does your project use TypeScript? No
 7// Where does your code run? browse
 8// What format do you want your config file to be in? JavaScript
 9// Would you like to install them now? Yes
10...

安装成功后 ESLint 帮我们创建了 .eslintrc.cjs 配置文件（cjs 是指 CommonJS 格式）,先不管，安装Prettier

Prettier

1yarn add prettier -D

然后再根目录创建 .prettierrc.cjs 配置文件

 1module.exports = {
 2  printWidth: 80,
 3  tabWidth: 2,
 4  useTabs: false,
 5  singleQuote: true,
 6  jsxSingleQuote: true,
 7  semi: false,
 8  trailingComma: "none",
 9  bracketSpacing: true
10}

需要注意：jsxSingleQuote: true 这条默认是false，如果设置为true，让jsx中使用单引号，就需要在ESLint中配置'prefer-single': 'on'开启，否则报错

ESLint + Prettier

接下来在 ESLint 中引入 Prettier，安装相关依赖。

1yarn add eslint-config-prettier eslint-plugin-prettier -D

再来配置下 .eslintrc.cjs

 1module.exports = {
 2  env: {
 3    browser: true,
 4    es2021: true
 5  },
 6  extends: [
 7    'eslint:recommended',
 8    'plugin:react/recommended',
 9    'plugin:prettier/recommended',
10    'plugin:react/jsx-runtime'
11  ],
12  parserOptions: {
13    ecmaFeatures: {
14      jsx: true
15    },
16    ecmaVersion: 'latest',
17    sourceType: 'module'
18  },
19  plugins: ['react', 'prettier'],
20  rules: {
21    'prettier/prettier': 'error',
22    'arrow-body-style': 'off',
23    'prefer-arrow-callback': 'off',
24    'no-undef': 'off',  
25    'no-unused-vars': 'off',
26    'react/prop-types': 'off'
27  },
28  settings: {
29    react: {
30      version: 'detect'
31    }
32  }
33}

这几个配置需要注意下，ESLine还是过分BT

1'no-undef': 'off', // 禁用未声明的变量，除非它们在 /*global */ 注释中被提到
2'no-unused-vars': 'off', // 禁止出现未使用过的变量
3'react/prop-types': 'off' // 禁用道具类型验证

Vite 中引入 ESLint

1yarn add vite-plugin-eslint -D

再来配置下vite.config.js，

 1import { defineConfig } from 'vite'
 2// import react from '@vitejs/plugin-react'
 3import reactRefresh from '@vitejs/plugin-react-refresh'
 4import path from 'path'
 5import viteEslint from 'vite-plugin-eslint'
 6
 7// https://vitejs.dev/config/
 8export default defineConfig({
 9  plugins: [
10    reactRefresh(),
11    viteEslint({
12      failOnError: false
13    })
14  ],
15  resolve: {
16    alias: {
17      '@': path.resolve(__dirname, '/src')
18    }
19  }
20})

以上配置基本完成。

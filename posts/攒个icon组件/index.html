攒个Icon组件

Simz
2022-07-28
react icon

经常会用到RemixIcon的图标，所以每次都得整半天，所以干脆攒个Icon的组件好了，下次直接用。

组件代码

因为依赖RemixIcon和styled-components，所以也需要安装这两个package。

 1import React from 'react'
 2import PropTypes from 'prop-types'
 3import styled, { keyframes } from 'styled-components'
 4import 'remixicon/fonts/remixicon.css'
 5
 6const rotate = keyframes`
 7  from {
 8    transform: rotate(0)
 9  }
10  to {
11    transform: rotate(360deg)
12  }
13`
14
15const IconWrapper = styled.span`
16  display: flex;
17  justify-content: center;
18  align-items: center;
19  width: ${(props) => (props.size || 16) + 'px'};
20  height: ${(props) => (props.size || 16) + 'px'};
21  font-size: ${(props) => (props.size || 16) + 'px'};
22  color: ${(props) => props.color || '#9e9e9e'};
23  animation: ${(props) => (props.spin ? rotate : null)} 1s linear infinite;
24`
25
26function Icon({ icon, type, size, color, spin }) {
27  return (
28    <IconWrapper size={size} color={color} spin={spin}>
29      <i className={`ri-${icon ? icon : heart}-${type ? type : line}`}></i>
30    </IconWrapper>
31  )
32}
33
34Icon.defaultProps = {
35  icon: 'heart',
36  type: 'line',
37  size: 16,
38  color: '#9e9e9e',
39  spin: false
40}
41
42Icon.propTypes = {
43  icon: PropTypes.string.isRequired,
44  type: PropTypes.oneOf(['line', 'fill']),
45  size: PropTypes.number,
46  color: PropTypes.string,
47  spin: PropTypes.bool
48}
49
50export default Icon

Porps

参数说明类型默认值
icon图标内容，见remixiconstringheart
type图标类型，包括：line | fillline | fillline
size图标大小number16px
color图标颜色string#9e9e9e
spin是否旋转booleanfalse
